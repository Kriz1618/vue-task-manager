# Firebase-vue
### Basic projects manager with VueJS and Firebase services
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Deploy

* Login to firebase
```
firebase login
```

* Init firebase selecting the hosting service
```
firebase init
```

* Build to compiles and minifies for production
```
npm run build
```

* Deploy 
```
firebase deploy
```
